   - Access control: Limited to team members

   - Personal details, contact info
   - Commission structures
   - Revenue splits
   - Access control: Private, shared only with team

5. **Competitive Research**
   - Competitor strategies, pricing, keyword targeting
   - May be sensitive if sharing externally
   - Access control: Internal only

### **Access Control Framework**

**Public (Can share widely):**
- General brand positioning
- Public rankings data
- Blog content strategy
- Press announcements

**Internal Only (Connor's team):**
- GSC raw data
- Artist economics
- Revenue/commission details
- Shopify configuration
- Specific competitor pricing

**External (Only with NDA):**
- Detailed competitive analysis
- Specific traffic numbers
- Artist personal information
- Negotiated on case-by-case basis

### **API Key & Credential Management**

**For Multi-Agent Work:**
- [ ] Store GSC credentials in environment variables (never in code)
- [ ] Use OAuth 2.0 for Shopify API access (not API keys)
- [ ] Use read-only API scopes where possible
- [ ] Rotate credentials quarterly
- [ ] Log all API access for audit trail
- [ ] Use Claude's built-in context isolation (agents can't see each other's prompts)

**For External Contractors:**
- [ ] Contractor access to read-only versions of data
- [ ] No direct Shopify admin access (share reports instead)
- [ ] NDA signed before sharing competitor analysis
- [ ] Time-limited access (revoke when contract ends)

### **Data Minimization**

When sharing with agents/contractors, share only what they need:
- Copywriter: Artist bios + brand guidelines (not analytics, not GSC data)
- Developer: Shopify theme files + product schema (not artist contracts)
- PR: Public brand story + media contacts (not revenue data)

## ü§ñ MULTI-AGENT ORCHESTRATION DESIGN

### **Agent Roles (For Ongoing Work)**

**Role 1: SEO Monitor Agent**
- **Responsibility:** Track rankings, flag changes, identify opportunities
- **Input:** GSC data, keyword list, competitor keywords
- **Output:** Weekly ranking reports, opportunity alerts
- **Trigger:** Weekly (Monday mornings) or on-demand
- **Tools:** Google Search Console API, web search
- **Security:** Read-only GSC access, no Shopify access

**Role 2: Content Planner Agent**
- **Responsibility:** Research topics, outline blog posts, suggest keywords
- **Input:** Blog topic list, artist bios, past performance data
- **Output:** Blog outlines, SEO keyword targets, editorial calendar
- **Trigger:** Monthly or on-demand
- **Tools:** Web research, document creation
- **Security:** Can see artist bios, cannot see artist contracts/revenue

**Role 3: Copywriting Coordinator Agent**
- **Responsibility:** Brief copywriters, review submissions, manage workflow
- **Input:** Brand guidelines, artist research, product info
- **Output:** Copywriter briefs, QA checklist, revision requests
- **Trigger:** On-demand (when copywriter is hired)
- **Tools:** Document creation, collaboration
- **Security:** Can see artist bios + brand guidelines, not contracts

**Role 4: Analytics Reporter Agent**
- **Responsibility:** Analyze trends, create dashboards, recommend actions
- **Input:** GSC data, traffic data, conversion data
- **Output:** Monthly reports, trend analysis, recommendations
- **Trigger:** Monthly or on-demand
- **Tools:** Google Search Console API, Google Analytics API
- **Security:** Read-only analytics access

**Role 5: Press Outreach Agent**
- **Responsibility:** Research publications, draft pitches, track outreach
- **Input:** Artist stories, press targets, brand story
- **Output:** Pitch emails, media contact list, outreach calendar
- **Trigger:** Monthly for Phase 2+, or on-demand
- **Tools:** Web research, email drafting, spreadsheet management
- **Security:** Public data only

**Risk: Phase 1 doesn't move rankings**
- *Cause:* Competitor moved up, or changes weren't indexed properly
- *Mitigation:* Check GSC Coverage tab, verify Shopify updates were published, wait another 2 weeks

**Risk: Blog doesn't generate traffic**
- *Cause:* Topics don't match search demand, or promotion is weak
- *Mitigation:* Research keywords first, amplify on social, guest post elsewhere
- *Fallback:* Pause blog, refocus on product page optimization

**Risk: Can't afford Phase 2**
- *Cause:* Budget constraints
- *Mitigation:* Do Phase 1 well first (highest ROI), pause Phase 2, restart when budget available
- *Fallback:* DIY blog (2 posts/month = 4-6 hours), focus on organic reach

**Risk: Team is too busy for ongoing content**
- *Cause:* Capacity constraints
- *Mitigation:* Start with 1 post/month (instead of 2), use agents to handle planning
- *Fallback:* Outsource to copywriter/agency retainer ($500-1000/month)

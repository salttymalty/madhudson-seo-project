Or: Push Everything Programmatically

The existing Node.js script at madhudson-phase1/ already writes to Shopify via GraphQL. It can be extended to push all of this in one run — titles, meta descriptions, and image alt text. The team just reviews and approves the data, then one command deploys it.

What the script already handles:

  • Product title updates via productUpdate mutation
  • Custom metafields (artist_name, artist_bio)

What needs to be added (straightforward):

  • SEO title + description: Add seo: { title, description } to the existing productUpdate mutation. Shopify docs
  • Image alt text: Use the fileUpdate mutation with each image's ID + new alt text. Shopify docs

Workflow:

  1. All the data from this paste sheet lives in a JSON file (already partially built at products-to-update.json)
  2. Connor reviews the JSON — approves or edits the text
  3. Run node src/index.js update-products
  4. Script pushes titles, meta descriptions, alt text, and metafields for all 6 products in one pass

This turns a 30-minute manual data-entry session into a review-and-approve workflow. The paste sheet becomes the approval document, not the data-entry document.

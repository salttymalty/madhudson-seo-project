Five things you can do this afternoon

None of these change how the site looks. All of them change how search engines see it.

  1. Add alt text to every product image Shopify Admin → Products → click each image → Alt text field
  2. Write custom meta descriptions for every product Products → Search engine listing preview → Description
  3. Update page titles to include artist names Same section → Page title field
  4. Enable bot filtering Analytics → Settings → "Exclude known bots"
  5. Add UTM parameters to link-in-bio ?utm_source=instagram&utm_medium=social&utm_campaign=linkinbio

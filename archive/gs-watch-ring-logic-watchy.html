01 — Ring Logic: Time Remaining in Day

TIMER ARC 6+ hrs 3-6 hrs 1-3 hrs NOW Depletion timer (user-set) Evening zone Afternoon zone Urgent zone

The Core Concept

Center = NOW. Rings = time remaining until end of waking day.

As the day progresses, outer rings "collapse" inward. By evening, only the innermost ring remains. This creates urgency awareness without clock-reading.

  • Outer ring (6+ hours): Plenty of day left. Relaxed.
  • Middle ring (3-6 hours): Afternoon. Day is progressing.
  • Inner ring (1-3 hours): Evening approaching. Limited time.
  • Center only (<1 hour): Day ending. Wind down.

Key insight: You don't need to know "it's 4:37pm" — you need to know "I have about 4 hours of day left."

Why This Works for Time Blindness

  • Spatial metaphor: "Lots of space" = lots of time. "Shrinking" = time running out.
  • No math required: Don't need to calculate "11pm minus 4:37pm = 6hr 23min"
  • Embodied urgency: Physical compression creates felt sense of day ending
  • Customizable day: User sets their wake/sleep times — rings calibrate automatically

03 — Haptic Cuckoo: Vibration Patterns

Like a cuckoo clock, the watch vibrates on the hour and half-hour. Pattern encodes time without looking.

Pattern Key

Long pulse (300ms)
Short pulse (100ms)
Gap (200ms)

Hour Encoding

Long pulse = hour marker
Short pulses = hour count (1-12)

User learns pattern over time. No visual check needed.

Haptic Options

  • Minimal: Vibrate on hour only (long pulse). Half-hour optional.
  • Encoded: Long + count pulses = actual hour (learn over time)
  • Ring transitions: Subtle buzz when a ring collapses (day progressing)
  • Timer warnings: Gentle pulse at 5min, 1min remaining. Distinct pattern at zero.
  • Quiet hours: Disable haptics during user-defined sleep window

Development Path

Phase 1: Build Watchy watch face with ring logic + timer bar. Test haptic patterns. Validate with 3-5 ND users.

Phase 2: Refine based on feedback. Add settings UI. Create 2-3 aesthetic variants (minimal, Girard color, dark mode).

Phase 3: If validated, explore mechanical hybrid (printed dial with ring zones + rotating bezel timer + haptic module).

Watchy is lowest-barrier path to test core concept before any hardware investment.

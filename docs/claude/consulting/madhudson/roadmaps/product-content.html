Product Content Guide

From ~30 words per product to 400-500 — with structure that works for SEO and answer engines

TL;DR

  • Current product pages have ~30 words of description — far below the 400-500 words that perform well
  • Each product needs 5 content blocks: Description, About the Artist, Design Intent, Specifications, Edition Details
  • Zero products currently have artist bios or edition information
  • Prioritize slow movers first: Dogma, Eddie, Marshall, Miami (566 units of inventory)
  • Automation scripts are built — write the content, scripts push to Shopify

Bottom line: The photos do the heavy lifting. The words need to catch up.

Writing Guide

▼

Practical guidance for each content block.

Product Description (100-150 words)

  • Open with the watch name and "limited edition artist collaboration watch"
  • Name the artist in the first 2 sentences
  • Describe the visual concept — what does the dial look like?
  • Include "Mad Hudson Watches" or "Mad Hudson" at least once
  • Include the price naturally: "At $[price], the [Watch] is..."
  • Include "Chicago" — geographic identity is a differentiator

About the Artist (100-150 words)

  • Who is the artist? Where are they based?
  • What medium do they normally work in?
  • What makes their style distinctive?
  • How did the collaboration with Mad Hudson come about? (if known)
  • What other notable work have they done?
  • Include the artist's full name at least twice for SEO

Design Intent (75-100 words)

  • What was the artist trying to express?
  • What materials or techniques are visible in the dial?
  • What cultural reference or concept drives the design?
  • This is the "why" behind the "what"

Specifications (50-75 words)

  • Use an HTML table or definition list
  • Include: Price, Movement, Case material + size, Crystal, Strap material + width, Water resistance, Designer name + city, Edition size
  • Structured format helps both search engines and comparison shoppers

Edition Details (25-50 words)

  • How many pieces in the edition?
  • Is it numbered? Signed?
  • What happens when they sell out? (Limited run vs. reissue)
  • Include "limited to [N] pieces" — this phrase ranks for scarcity queries

Shopify Implementation

▼

Where these content blocks go.

  • All content goes in the Shopify product description field (Products → [Product] → Description in the rich text editor)
  • Use H2 headings for each section: "About the Artist," "Design Intent," "Specifications," "Edition Details"
  • Specs should use an HTML table, not prose paragraphs
  • Don't hide content in JavaScript tabs or accordions — search engines may not see it
  • The automation scripts can push all content at once: write in JSON template, validate, dry-run, deploy
  • Before deploying: duplicate the current theme as a backup

Using the Automation Scripts

▼

Brief overview of the deployment workflow.

  1. Write content in the JSON template file
  2. Run validation: checks word counts, meta description length, placeholder detection
  3. Run dry-run: see before/after for every field
  4. Deploy: one command pushes all 6 products at once
  5. Alt text for images is handled separately by the image automation script

Note: The automation scripts are built and tested. The bottleneck is writing the content — not the deployment. Focus writing time on slow movers first.

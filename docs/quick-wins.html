Five same-day fixes

None change how the site looks. All change how search sees it. Start with what's fastest or what matters most to you.

Win 1

Add alt text to every product image

Products → click image → Alt text
Search engines can't see images without alt text. Every product image is currently invisible to Google.
Win 2

Write custom meta descriptions

Products → Search engine listing → Description
Shopify auto-generates these from page content. Custom descriptions control what appears in search results.
Win 3

Update page titles with artist names

Products → Search engine listing → Page title
Artist-collaboration is what differentiates Mad Hudson. Page titles should reflect that.
Win 4

Enable bot filtering

Analytics → Settings → Exclude known bots
Bot traffic inflates session counts and suppresses conversion rate metrics.
Win 5

Add UTM parameters to link-in-bio

?utm_source=instagram&utm_medium=social&utm_campaign=linkinbio
62% of revenue has no tracked source. UTM parameters close the attribution gap.

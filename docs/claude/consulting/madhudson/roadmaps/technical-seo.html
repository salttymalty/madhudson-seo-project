Technical SEO Checklist

Same-day wins and structural fixes that help Google see what's already there

TL;DR

  • Most of these fixes are invisible to customers — the site looks exactly the same
  • 5 same-day wins can be done in a single afternoon in Shopify admin
  • Page titles, meta descriptions, and alt text are the highest-impact low-effort fixes
  • Bot filtering reveals your true conversion baseline (~0.85% vs reported 0.06%)
  • UTM parameters solve the $28K attribution blindspot within one month

Bottom line: These are the foundation. Everything else — content, schema, AEO — builds on top of clean technical SEO.

Page Titles

▼

Current format: [Watch Name] — Mad Hudson Watches (Shopify auto-generated)

Target format: [Watch Name] by [Artist] — Limited Edition Artist Watch | Mad Hudson

Page Type Current Title Recommended Title
Product pages [Watch Name] — Mad Hudson Watches [Watch Name] by [Artist] — Limited Edition Artist Watch | Mad Hudson
Homepage Mad Hudson Watches Mad Hudson Watches — Artist Collaboration Watches from Chicago
Collections [Collection] — Mad Hudson Watches [Collection] — Artist-Designed Limited Edition Watches | Mad Hudson

How to change

Shopify admin → Products → [Product] → scroll to "Search engine listing preview" → edit "Page title" field. No theme editing required.

Why it matters: Page title is the most prominent text in search results. Including artist names captures searches like "[Artist Name] watch." Adding "limited edition" and "artist" captures category queries.

Example: Instead of "The CD Watch — Mad Hudson Watches", use "The CD Watch by Riki Prosper — Limited Edition Artist Watch | Mad Hudson". This captures searches for "Riki Prosper watch", "artist collaboration watch", and "limited edition watch" all at once.

Meta Descriptions

▼

Current state: Auto-generated from the first ~160 characters of thin product descriptions. Result: "A beautiful timepiece from Mad Hudson Watches." — generic, keyword-poor.

Template:
[Watch Name] — a limited edition watch designed by [Artist Name] for Mad Hudson. [One sentence about the design]. [Edition size] pieces. Free shipping.

Rules

  • Aim for 150-160 characters (Google's display limit)
  • Include artist name, edition info, "limited edition" or "artist collaboration"
  • Every product and the homepage need custom meta descriptions
  • Front-load the most important info (artist, watch name, key detail)

How to change

Same "Search engine listing preview" section in Shopify admin → Description field.

Before/After Comparison

Before
"A beautiful timepiece from Mad Hudson Watches."
After
"The CD Watch — a limited edition wearable tribute to mixtapes, designed by Atlanta artist Riki Prosper. Holographic disc dial. $215. Free shipping."

Impact: Better CTR from search results. The "after" version tells the searcher exactly what makes this watch special, who designed it, and why it's worth clicking.

Alt Text

▼

Current state: Empty or auto-generated filenames on all product images.

Image Type Current Alt Text Recommended Alt Text
Product hero shot "" (empty) or filename [Watch Name] by [Artist] — [brief visual description] — Mad Hudson Watches
Detail / dial close-up "" (empty) Close-up of [Watch Name] dial showing [specific design element] by artist [Name]
Wrist shot / lifestyle "" (empty) [Watch Name] artist collaboration watch worn on wrist — limited edition by [Artist]
Artist portrait N/A (doesn't exist yet) Artist [Name], designer of [Watch Name] for Mad Hudson Watches

How to change

Products → click each image → Alt text field. Budget 2-3 minutes per image.

Automation option: Alt text can be updated for all images in one script run — provide descriptions in a JSON template.

Why it matters: Opens Google Image Search traffic (currently zero from this channel). "Artist collaboration watch," "limited edition art watch," and "[artist name] watch" are all image-searchable queries. Also improves accessibility.

Quick win: Alt text takes 20-30 minutes for all product images and has immediate impact. Google Images often indexes faster than web search.

Heading Hierarchy (H1-H6)

▼

Common Issues

  • Multiple H1 tags per page (theme uses H1 for both logo and product name)
  • Skipped heading levels (H1 → H3, skipping H2)
  • No H2 sections for content blocks (About the Artist, Specifications, Edition Details)

Target Structure

  • One H1 per page (the product name)
  • H2 for major sections (About the Artist, Specifications, Edition Details)
  • H3 for subsections (Materials, Care Instructions, Shipping)

How to check

Right-click any product page → View Page Source → search for <h1, <h2, etc.

How to fix

Theme editor → product template. Change extra H1 tags to appropriate levels. This is a one-time fix.

Why it matters: Search engines use heading hierarchy to understand page structure. Clean hierarchy helps Google extract featured snippets and better understand what's most important on the page.

Image Optimization

▼

Filenames

Current pattern: Generic (IMG_4532.jpg). Existing images can't be renamed in Shopify.

Future uploads: mad-hudson-[watch-name]-[artist]-[view].jpg

Example: mad-hudson-cd-watch-riki-prosper-dial.jpg

Note: Descriptive filenames are a minor ranking signal, but they help with organization and make alt text easier to write. Don't rename existing images — just follow the convention for new uploads.

Format

Shopify's CDN handles compression and WebP conversion automatically. No action needed.

Size

Upload at 2048px on the longest edge. Shopify generates all thumbnails. Don't upload 5000px+ originals — slows down admin.

Watch out: Uploading enormous files (5000px+, 10MB+ each) can slow down Shopify admin and doesn't improve display quality — the CDN will resize them anyway.

URL Structure

▼

Shopify handles URLs well by default (/products/[handle], /collections/[handle]).

Main Opportunity

Ensure product URL handles include artist name where they don't already. URL handles can be edited in Shopify — set up redirects if changing.

Example:

  • Current: /products/the-cd-watch
  • Better: /products/cd-watch-riki-prosper (if changing, redirect the old URL)

How to change

Products → [Product] → scroll to "Search engine listing preview" → click "Edit website SEO" → change URL handle.

Important: After changing, add a redirect: Navigation → URL Redirects → map old URL to new URL.

Caution: Only change URLs if the product hasn't been heavily promoted yet. Broken links lose SEO equity and frustrate customers. For existing watches with inbound links, keep the current URLs and use page titles/meta descriptions to include artist names instead.

UTM Tracking

▼

The $28.3K problem: 179 orders (62% of total revenue) have no tracked source.

Fix

Add UTM parameters to Instagram link-in-bio:

https://madhudsonwatches.com?utm_source=instagram&utm_medium=social&utm_campaign=linkinbio

Also add to any email campaign links:

https://madhudsonwatches.com/products/cd-watch?utm_source=email&utm_medium=newsletter&utm_campaign=[campaign-name]

Timeline

Within one month of implementing, you'll know what the unattributed bucket actually is. If Instagram is the real #1 channel, the entire strategy shifts.

Effort

10 minutes — Shopify tracks UTM parameters automatically. Just update the link-in-bio URL.

Immediate value: UTM tracking costs nothing and solves one of the biggest unknowns in the entire dataset. Do this today.

Crawl Verification

▼

Ensure Shopify theme serves product content as crawlable HTML, not JavaScript-only content. Some Shopify themes render product descriptions via client-side JS, which search engines and answer engines may not execute.

How to check

Right-click a product page → View Page Source (not Inspect Element). Search for your product description text. If it's in the source, it's crawlable. If it only appears after JS runs, it's not.

What to Do if JS-Only

  • Contact Shopify support or switch to a theme that server-renders content
  • Most modern Shopify themes (Dawn-based) server-render by default

Good news: Most Shopify themes handle this correctly. This is a one-time check to confirm, not a likely problem.

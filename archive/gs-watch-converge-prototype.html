Converged Prototypes

A. Depletion Arc

Candle clock inheritance

Timer Mode
remaining

Mechanism

Arc depletes as time is consumed. Material = time. When arc is gone, duration is over.

Cognitive Model

Bounded ownership. "This is my time, I'm using it." No infinite tracking — just what you have.

  • Strong timer metaphor (candle, water clock)
  • Clear start/end boundary
  • Works for Pomodoro, meetings, tasks
  • E-ink: 12 discrete steps sufficient
  • Weakness: Less useful for "what time is it"

B. Radial Now

Expanding circle from center

Clock Mode

Mechanism

Center = now. Rings expand outward = future hours. Edge = day's horizon. You are the center, not peripheral.

Cognitive Model

Embodied "here" concept. Future approaches you rather than you chasing time. Inverts traditional clock anxiety.

  • Strong spatial grounding (near/far)
  • Day overview at a glance
  • Works for situating in day
  • E-ink: Concentric rings easy to render
  • Weakness: Less precise for exact time

C. Block State

Discrete time-of-day zones

Hybrid

Mechanism

5-6 day blocks (dawn, morning, midday, afternoon, evening, night). Active block highlighted. Dot grid shows progress within block.

Cognitive Model

Ambient awareness via color. "What part of day am I in?" vs "What exact time?" Aligns with incense clock multi-sensory approach.

  • Best for time blindness (discrete states)
  • Ambient time-feel without reading
  • Dot grid adds precision without numbers
  • E-ink: Perfect fit (5-6 states)
  • Weakness: Cultural time blocks vary

Converge Recommendation

Primary direction: C. Block State — Best balance of timer/clock, strongest time blindness support, highest differentiation. Secondary: A. Depletion Arc as dedicated timer mode. Consider B. Radial as "calm mode" variant within Block State system.

External Validation Required

Before further development

  1. User testing: Show prototypes to 3-5 people with ADHD/time blindness. Which resonates? Which confuses?
  2. Mechanism check: Does "Block State" actually help time-blind users, or does it just feel intuitive to designer?
  3. Existing research: Any studies comparing visual timer formats for ND users?
  4. Cultural blocks: Are 5-6 day divisions universal enough, or too culturally specific?
